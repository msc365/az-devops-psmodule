# Get-AdoPolicyType

## SYNOPSIS

Gets policy types for an Azure DevOps project.

## SYNTAX

### __AllParameterSets

```text
Get-AdoPolicyType [-ProjectId] <string> [-PolicyType] <string> [[-ApiVersion] <string>]
 [<CommonParameters>]
```

## ALIASES

- N/A

## DESCRIPTION

This function retrieves policy types for an Azure DevOps project through REST API.

## EXAMPLES

### EXAMPLE 1

#### PowerShell

```powershell
Get-AdoPolicyType -ProjectId 'my-project' -PolicyType '00000000-0000-0000-0000-000000000000'
```
